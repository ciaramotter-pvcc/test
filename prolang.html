Definition
Machine language The lowest level programming language that computers use to understand and execute instructions.
Debugging A process in which an error is located and corrected.
Assembler Programs developed to convert mnemonic expressions into machine language instructions.
Assembly language A mnemonic system for representing programs.
Translator Turns translated programs into machine-language programs.
Compiler A translator that often has to compile several machine instructions into short sequences to simulate the activity requested by a single high-level primitive.
Interpreter An alternative to translateors. They executed the instuctions as they were translated instead of recording the translated version.
Cross-Platform Software Programs that are operating-system independent.
Natural Language A language that has developed naturally in use, unlike formal language.
Formal Language Defined by grammars, often used with a serious tone.
Machine independence Theoretically a third-generation language can be used on any machine.
High-level programming language Provides more abstractions for the programmer and make it easier for people to read and write a program.
Object-oriented programming (OOP) An action in programming where a collections of units, called objects, each of which is capable of performing the actions that are immediatly related to itself as well as requesting actions of other objects. These objects interact to solve the problem at hand.
Scripting language A subset of the imperative programming languages.
Data types
  • Integer: numeric data consisting of whole numbers, probably stored using two's complement notation.
  • Float: Refers to numeric dada that might contain values other than whole numbers, probably stored in floating-point notation.
  • String: An array data structure of bytes that stores a sequence of elements, typically characters, using some character encoding.
