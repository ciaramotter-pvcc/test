GitHub Q & A

What is a version control system?

  • It is an important software development practice for tracking and managing changes made to code and other files.

    • What is Git?

  • Git is a mature, actively maintained open source project originally developed in 2005 by Linus Torvalds.

    • What is Github?

  • GitHub is a web-based interface that uses Git, the open source version control software that lets multiple people make seperate changes to web pages at the same time.

    • What is the difference between Git and Github?

  • Git is a version control system that relies on itself, while Github runs on Git.

    • Who started Github and how was it started?

  • Chris Wanstrath, P.J. Hyett, Tom Preston-Werner, and Scott Chacon were the original service delevopers. They used Ruby on Rails to develop it.

    • What company owns it now?

  • Microsoft Corporation has owned GitHub since 2018.

    • How much does a Github account cost?

  • An individual account is free, Team costs $4 per user/month, and Enterprise costs $21 per user/month.

    • What is the Octocat?

  • It is Githubs mascot.
    • Git/Github terms Definition
    Repository Repositories are where you can store your code, your files, and each file's revision history.
    Commit You can save small groups of meaningful changes as commits
    Fork A fork is a new repository that shares code and visibility settings with the original "upstream" repository.
    Push The git push command is used to upload local repository content to a remote repository.
    Pull requests A Pull request is a proposal to merge a set of changes from one branch into another
    workflows A Workflow is a configurable automated process that will run one or more jobs.
    Issues Issues can track bug reports, new features and ideas, and anything else you need to write down or discuss with your team
    Raw button A button that will let you into Raw view, you can view or copy the raw content of a file without any styling
    Blame Button The button is used to examine the contents of a file line by line and see when each line was last modified and who the author of the modification was.
