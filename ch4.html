Item Definition
Network Classifications
LAN A computer network that normally consists of a collection of computers in a single building or building complex. Examples include university campuses or manufacturing plants.
MAN A computer network that has an intermediate size, one that can span a local community.
WAN A computer network that can link machines over a greater distance. They can link machines in neighboring cities or opposite sides of the world.
Network Models
Bus Network A topology of a network in which machines are all connected to a common communication line.
Star Network A topology of a network where a single machine serves as a focal point to which all the others are connected.
Protocols
CSMA/CD Ethernet This protocol dictates that each message be broadcast to all the machines on the bus. It is not compatible with star networks.
CSMA/CA Wifi A protocol where a wireless network adopts the policy of trying to avoid collisions rather than trying to detect them.
Combining Networks
Repeater A divice that that passes signals back and forth between the two original buses without considering the meaning of signals.
Bridge A divice like the repeater, connects two buses, but it does not necessarily pass all messages across the connection. A destination address comes with the message and is forwarded to a machine that has relation to the address.
Switch A bridge with multiple connections, allowing it to connect to several buses rather than just two.
Router A divice that provides links between networks while allowing each network to maintain its unique internal characteristics.
Gateway A passage way between a network and the outside world.
Client/server model
P2P: peer to peer A client/server model that involves one process (the server) providing a service to and recieve service from each other.
Distributed sytem A client/server model that consist of softwar units that execute as processes on different computers.
Cloud Computing A client/server model is where huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing
Tier 1 ISPs that consist of very high-speed, high-capacity, international WANs.
Tier 2 ISPs that tend to be more regional in scope and less potent in their capabilities.
Tier 3 An access ISP that is essentially an independent internet, sometimes called an intranet, operated by a single authority that is in the business of supplying internet access to individual homes and businesses.
IP address
ICANN A nonprofit corporation established to coordinate the internet's operation.
DNS An internet-wide dictionary system that collectivley consist of name servers that converts a mnemonic address into a IP address.
TLD: Top-Level Domain The suffixes that are at the end of a domain name.
Country-code TLDs Two letter TLDs made for specific countries.
Internet Protocols
FTP An application for listing and copying files across the network.
HTTP Handles webservers and browsers.
SMTP Defines a way that two computers on the network may interact when transmiting an email message from one host to the other.
World-wide web
URL A unique address given to a document that contains information needed by the browser to contact the proper server and request the desired document.
Client-side activities An activity that is performed by a client. (such as a browser)
Server-side activities An activity that is performed by a server. (such as a webserver)
Internet four-layer software
Application layer Where the message originates from. It uses the transport layer to send and recieve messages over the internet.
Transport layer Accepts messages from the application layer and ensures that the messages are properly formatted for transmission over the internet.
Network layer Decides in which direction a packet should be sent at each step along with the packet's path through the internet.
Link layer Transfers the packet and deals with the communication details particular to the individual network in which the computer resides.
Internet is:
Packet-switching Network A method of transferring data to a network in the form of packets.
Fault tolerant Allows an system to maintain proper operation despite failure or faults in one or more of its components.
Redundant Something that is not needed
Cybersecurity
Malware malicious softwares that can be on a computer itself or from a computer from a distance.
Virus A software that infects a computer by inserting itself into programs already inside the machine.
Worm An autonomous program that transfers itself though a network, taking up residence in computer and forwarding itself to other computers.
Trojan Horse A program that enters a computer system desguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.

Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
Phishing A technique of obtaining information explicity by simply asking for it.
DDos attack An attack made for multiple hosts by generating a large amount of messages in a short period of time.
Protection
Firewall A program that filters traffic passing into a point in the network.
Spam filter A variation of a firewall that is designed to block unwanted email.
Proxy server A software unit that acts as an intermediary between a client and a server with a goal of shielding the client from adverse actions of the server.
Antivirus software A softwre that detects and removes the presence of known viruses and other infections.
Cryptography The study of sending and recieving messages securely in the presence of adversaries.
Public key encryption Encrypts and decrypts messages.
