Operating Systems Terminology

Term Definition
Operating System A system software that manages computer hardware and software resourses
Job queue A queue where jobs await in a FIFO format to get executed.
FIFO FIFO is an order in which a job is completed, the first job that arrives is completed first.
Interactive processing It allows you to be able to see and change what is on the computer from a remote location.
Time-Sharing It allows several users to use the same computer at the same time without reducing productivity.
Multiprogramming Multiprogramming is when time is turned into intervals, the execution of each job is then restricted to one interval at a time.
Application software Application software consists of the programs for performing tasks particular to the machine's utilization.
System software The system software provides the infrastructure that the application software requires.
Utility software Utility software perform activities that are fundamental to computer installations. They don't come with the operating system.
Graphical user interface (GUI) The GUI is what allows a user to interact with the computer with visual imputs.
Shell Shell is an old version of a user interface, it communicates with users through textual messages.
Kernal The internal part of an operating system is the kernal, it contains software components that perform basic functions that are required by the installation.
Device driver A part of the kernal that carry out operations and communicate with controllers and/or peripheral devices.
File manager A part of the kernal that coordnates the use of the mass storage units.
Memory manager A part of the kernal that regulates usage of the machine's main memory.
Virtual memory Memory created on a disc to reserve for tasks that need more memory of what's availiable.
Boot Strapping A procedure that boots up the operating system by moving it from mass storage to the main memory.
ROM A memory that can be read, but cannot be altered.
Deadlock A error in wich two or more processes are blocked from processing because each is waiting for a resource that is allocated to another.
Spooling Spooling is when data is held for a later but more convenient time.
Super user (administrator) The admin has special access and privleges within a operating system. eg: changing settings.
Multi-factor authentication Mfa is a way to secure accounts and devices by many means. You can use SMS authentication, email authentication, and a question that relates to the user.
